# Typescript meetup

Typescript meetup examples

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/typescriptMeetup.git
```

Install packages

```bash
npm install
```

transpile ts files

```bash
npm run build
```

execute directly ts file

```bash
npm run test <pathFile>
```

clean dist directory

```bash
npm run clean
```
